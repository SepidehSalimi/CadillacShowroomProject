# پروژه سایت نمایشگاه ماشین

این پروژه با استفاده از ASP.NET Core MVC طراحی شده و یک سیستم مدیریت خرید و فروش خودرو را شبیه‌سازی می‌کند.

##  قابلیت‌ها

-درخواست کارشناسی توسط کاربران به مدیریت
- جستجوی پیشرفته بر اساس برند، مدل، قیمت و ...
- مشاهده‌ی جزئیات هر خودرو به همراه تصویر
- پنل مدیریت برای تأیید، ویرایش و حذف آگهی‌ها توسط مدیر گالری
- استفاده از معماری سه‌لایه (Three-Layer Architecture)

##  تکنولوژی‌های استفاده‌شده

- C#  
- ASP.NET Core MVC  
- Entity Framework  
- SQL Server
- HTML
- CSS
- JS
- Bootstrap (برای طراحی ظاهری)
